# Base layout template with Gulp, SCSS and Stylelint
1. A simple and engaging implementation of the classic 2048 puzzle game, where players combine tiles with matching numbers to reach the legendary 2048 tile. This project demonstrates core game logic, keyboard interactions, and smooth UI updates.
2. - [DEMO LINK](https://Mariiaantoniv.github.io/2048_Saga/)
3. HTML5 — the stage where everything happens.
It lays out the 4×4 battlefield, gives every tile its own little square to live in, and provides the structure your JavaScript brain can manipulate. Without HTML, your game would just be… floating numbers in the void.

  CSS3 — the stylist and animator of the whole experience.
It’s responsible for the colors, the tile themes, the satisfying “bigger number = stronger glow” effect, and the overall vibe that makes the game feel alive. CSS is what turns raw logic into something you actually want to play.

  JavaScript — the mastermind behind the scenes.
This is where the real action happens:

 - It listens for your arrow keys,
 - Slides tiles in the right direction,
 - Merges matching numbers with perfect timing,
 - Spawns new tiles in random spots,
 - Tracks your score like a loyal accountant,
 - Detects wins, losses, and everything in between,
 - And updates the UI so fast it feels like magic.
4. Features
  -  Smooth tile movement and merging
  -  Random tile generation (2 or 4)
  -  Score tracking
  -  Win and Game Over detection
  -  Responsive and clean UI
  -  Keyboard controls
  -  Start / Restart button
  -  Fully encapsulated game logic inside a Game class
